# Smart Brain

## To run project:

1. Clone this repo
2. Run `npm install`
3. Run `npm start`

## Overview of Project
- Smart Brain is a fullstack web app that uses the Clarifai API to locate a face in a (PNG/JPG) image link.
- Features include: 
  - Profile Registration Form
  - Sign-In/Sign-Out
  - Face Detection Model
  - Total Entry Count
